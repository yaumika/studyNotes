css-字体属性

text-align 文本位置

text-align 文本对齐，居中

user-align:right; 右对齐

user-align:left; 左对齐，文本一般默认左对齐


text-decoration 文本装饰

text-decoration: overline; 上划线

text-decoration: underline; 下划线

text-decoration: line-through; 删除线

text-decoration: none; 一般使用在链接标签，去除下划线

text-indent :文本缩进

首行缩进2个字符的大小 text-indent: 2em;

或者指定缩进大小 text-indent: 20px;

也可以使用缩进负值，使每段首行向左移动


line-height 定义行间距

测量屏幕内的像素值：FSCapture

设置字体使用 font 系列的

font-family 可以添加多种字体，以防单一字体不兼容

font-family: 'Times New Roman', Times, serif;

多种字体，优先使用最前面的单一句font时

font-size 可以改变字体大小

font-size: 20px;

谷歌浏览器默认字体大小为16px

标题的字体一般优先使用默认字体大小 如果要改变需要指定修改

font-weight 字体的粗细、

font-weight: 700 或 font-weight: bold

实际开发需要使用数字进行编辑字体粗细 400 == normal 700 == bold

normal 可以将默认加粗的字体取消加粗

此处为一个h4标签，但是经过了normal处理

font-style 改变字体的风格

font-style: italic;将字体变为倾斜状态

font-style: normal;将字体从倾斜状态变为正常

复合字体属性

font-style: italic;

font-weight: 700;

font-size: 20px;

font-family: 'Times New Roman', Times, serif;

可以复合上述代码为一句：

font: italic 700 20px "Times New Roman", Times, serif;

顺序必须遵循顺序 font: font-style font-weight font-size/line-height 和 font-family;

必须含有 font-size/line-height 和 font-family两个属性 否则不起作用


color 颜色的三种表现形式：

预置颜色：red blue green pink

十六进制表示：#fff000

rgb即三原色red green blue：(255,0,255)

rgba 三原色之外 a表示透明度


css的三种引入方式

内部样式表：

即当前页面的最前面的style标签内的内容

单独控制一个页面

行内样式表:

即在标签内写 style="" 添加属性

仅可控制当前的标签

外部样式表:

单独写一个外部样式的表，.css结尾

使用link标签引入css文件

<link rel="stylesheet" href="css文件路径">

可以控制多个页面
